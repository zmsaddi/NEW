# MSADDI Web

Responsive multilingual website for MSADDI Sheet Metal Fabrication Service.

## Tech Stack
- Next.js & TypeScript
- Tailwind CSS
- i18next (EN + AR, extensible)

## Getting Started

```bash
# Install dependencies
npm install

# Run locally
npm run dev
```

## Deployment
Push to GitHub. Vercel will detect Next.js and build automatically.
